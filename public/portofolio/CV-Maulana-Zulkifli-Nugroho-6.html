4. Angkasa Travel
Angkasa Travel is a website & application for people who are currently searching for a flight ticket.
Angkasa Travel was built using NextJS, TailwindCSS, and other tools. Also I create the Back End
which is built with Go, Gorm, PostgreSQL, and Redis.
